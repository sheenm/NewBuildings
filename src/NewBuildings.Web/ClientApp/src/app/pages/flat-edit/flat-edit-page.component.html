Страницы объекта.

На странице отображаем форму для редактирования объекта. На форму выводим все характеристики (параметры) объекта Для редактирования доступны следующие параметры:
  • Площадь общая
  • Площадь кухни
  • Этаж
  • Стоимость, тыс. руб.
На странице выводим две кнопки: Сохранить. По клику сохраняем изменения объекта, и возвращаемся на главную страницу. При этом все внесенные в объект изменения отображаются в сетке квартир. Отмена. По клику изменения объекта не сохраняем, и возвращаемся на главную страницу.
